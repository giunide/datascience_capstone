Background and Model

This application was created for the Coursera Data Science Specialization Capstone.

This project involved developing a predictive model of text starting with a really large, unstructured database of the English language. In order to do this, it was necessary to clean the data while doing an exploratory analysis to understand what we were dealing with.

To be specific, all text was set to lower case, and the punctuation symbols were removed, as the numeric characters, and the profanity words.

Later, we were tasked with building a prediction model and testing it, in pursuance of making a data product in the shape of a Shiny Application

The model used was a simple “Backoff Model”, which is constructed using N-Grams and the Markov assumption, which in this context states that the probability of a word depends only on the previous word or sequence of words. For this project I built an unigram, bigram, trigram, and quadgram frequency dictionaries. When the user submits some text, the algorithm will take the last three words and look for them in the quadgram dictionary, if there’s a match it’ll give the prediction, else, it will take last two word and search in the trigram dictionary, and so on until it reaches the unigram dictionary.

Application

My Application has three grids:

  1. Input grid (Upper-Left Corner): Where the user specifies the text which it’s next word will be predicted, and the desired number of suggestions.

  2. Table grid (Upper-Right Corner): Where a table displays the predicted words and their associated probability.

  3. Interactive plot grid (Bottom): Where a bar plot displays the predicted words and their associated probability.
